О программе

Разработанное приложение предназначено для наглядного представления работы циклического алгоритма планирования процессов.
Для наглядного представления пользователю необходимо добавить процесс, а для этого нужно заполнить поля:
• Название – название процесса.
• Время выполнения – время, необходимое для выполнения данного процесса.
• Время переключения – время контекстного переключения.
Разработанное приложение также отображает графически работу алгоритма Round Robin. Графическое представление разработано в виде прямоугольников, которые выстраиваются на форме. Прямоугольники представляют собой процессы, внутри прямоугольников указывается оставшееся время до выполнения процесса.
Программный продукт реализован в виде приложения Windows Forms. Программа выполнена в среде Microsoft Visual Studio Community 2015, разработана в операционной системе Windows 10. Код программы написан на языке программирования C#. Программа проста в использовании и не требует специальных знаний у пользователя.
