Добавить процесс

Для того, чтобы добавить новый процесс необходимо заполнить все поля данными.
Поле «Название» необходимо заполнять только буквенными символами, но в любом случае, приложение просто не позволит пользователю вводить иные символы, это касается всех текстовых полей данной программы.
Поле «ID» заполняется автоматически, это поле, которое содержит порядковый номер процесса.
Поле «Время выполнения» - это время, необходимое для выполнения создаваемого процесса, поле заполняется цифрами, и время подразумевается в миллисекундах.
Поле «Время переключения» - это время переключения контекста, данное поле также заполняется цифрами, но, следует обратить внимание, что в данном поле вводится процентная составляющая от времени выполнения.
Если пользователь оставит незаполненным хоть одно поле, то система выдаст предупреждение.
